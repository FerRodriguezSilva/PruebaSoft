< h1-h6 > Los elementos HTML <h1> a <h6> representan seis niveles de encabezados de sección. <h1> es el nivel de sección más alto y <h6> es el más bajo El uso de diferentes niveles de encabezados ayuda al escaneo de la pagina y la facil lectura del mismo ayuda a los usuarios a navegar por la pagina
< p > El elemento HTML <p> representa un párrafo Los párrafos generalmente se representan en los medios visuales como bloques de texto separados de los bloques adyacentes por líneas en blanco y/o sangría en la primera línea, pero los párrafos HTML pueden ser cualquier agrupación estructural de contenido
<br> El elemento HTML <br> produce un salto de línea en el texto Es útil para escribir un poema o una dirección, donde la división de líneas es significativa
<hr> El hr representa una transicion entre parrafos con un efecto visual Se usa para separar temas relacionados pero separados
<wbr> Indica al navegador un salto de linea para ajustar el texto una posición dentro del texto donde el navegador puede opcionalmente dividir una línea, aunque sus reglas de división de línea no crearían de otro modo una ruptura en esa ubicación
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos Los superíndices generalmente se representan con una línea de base elevada usando texto más pequeño
<sub> Especifica el texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos Los subíndices generalmente se representan con una línea de base más baja usando texto más pequeño
<strong> Indica que su contenido tiene una gran importancia, seriedad o urgencia Los navegadores normalmente muestran el contenido en negrita
<em> Marca el texto que tiene énfasis El elemento <em> se puede anidar, y cada nivel de anidamiento indica un mayor grado de énfasis
<mark> representa texto que está marcado o resaltado para fines de referencia o notación La relevancia del pasaje marcado en el contexto que lo contiene
<q> indica que el texto adjunto es una cita breve en línea. La mayoría de los navegadores modernos implementan esto encerrando el texto entre comillas Este elemento está diseñado para citas cortas que no requieren saltos de párrafo
<blockquote> indica que el texto adjunto es una cita extendida. Por lo general, esto se representa visualmente mediante sangría Se puede dar una URL para la fuente de la cita usando el atributo cite, mientras que una representación de texto de la fuente se puede dar usando el elemento <cite>
<cite> Se utiliza para marcar el título de un trabajo creativo citado La referencia puede estar en forma abreviada de acuerdo con las convenciones apropiadas al contexto relacionadas con los metadatos de citas
<dfn> Identifica el termino a definir en el texto mientras que la definicion del termino se coloca fuera del elemento se utiliza para indicar el término que se define dentro del contexto de una frase u oración de definición
<abbr> Representa una abreviatura o un acrónimo Cuando incluya una abreviatura o un acrónimo, proporcione una expansión completa del término en texto sin formato la primera vez que lo use, junto con <abbr> para marcar la abreviatura. Esto informa al usuario qué significa la abreviatura o sigla
<small> De forma predeterminada, representa el texto dentro de un tamaño de fuente más pequeño, como de pequeño a x-pequeño representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independientemente de su estilo de presentación
<ins> Marca las partes que han sido agregadas al documento Puede usar el elemento <del> para representar de manera similar un rango de texto que se ha eliminado del documento
<del> Representa un rango de texto que se ha eliminado de un documento Esto se puede usar al representar "cambios de seguimiento" o información de diferencias de código fuente,
<s> muestra el texto tachado o con una línea que lo atraviesa Usa el elemento <s> para representar cosas que ya no son relevantes o precisas. Sin embargo, <s> no es apropiado para indicar ediciones de documentos
<code> Muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora De forma predeterminada, el texto del contenido se muestra con la fuente monoespaciada predeterminada del agente de usuario
<samp> Su contenido generalmente se representa utilizando la fuente monoespaciada predeterminada del navegador (como Courier o Lucida Console) se utiliza para encerrar texto en línea que representa la salida de muestra (o citada) de un programa de computadora
<kbd> Representa un tramo de texto en línea que indica la entrada textual del usuario desde un teclado, entrada de voz o cualquier otro dispositivo de entrada de texto Por convención, el agente de usuario presenta de forma predeterminada el contenido de un elemento <kbd> utilizando su fuente monoespaciada predeterminada, aunque el estándar HTML no lo exige
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML El texto normalmente se representa con una fuente no proporcional o monoespaciada. Los espacios en blanco dentro de este elemento se muestran como están escritos
<address⪧ Indica que el HTML adjunto proporciona información Información de contacto para una persona o personas, o para una organización
<div> Es el contenedor genérico para el contenido de flujo No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica estilo de alguna manera usando CSS
<table> Representa datos tabulares Es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos
<thead> Definen el encabezado de las columnas de la tabla Se usa para la cabecera de una tabla para mostrar sus titulos o encabezados principales
<tbody> Encapsula un conjunto de filas de tabla Lo que indica que comprenden el cuerpo de la tabla
<tfoot> Define un conjunto de filas que resumen las columnas de la tabla Se usa para el pie de la tabla algun resultado final o un cierre a la tabla
<tr> Define una fila de celdas en una tabla Luego, las celdas de la fila se pueden establecer mediante una combinación de elementos
<th> Define una celda como el encabezado de un grupo de celdas de tabla La naturaleza exacta de este grupo está definida por los atributos de alcance y encabezados
<style> Contiene información de estilo para un documento o parte de un documento Contiene CSS, que se aplica al contenido del documento que contiene el elemento <style>
<head> Contiene información legible por máquina (metadatos) sobre el documento Como su título, scripts y hojas de estilo
<header> Representa contenido introductorio, normalmente un grupo de ayudas introductorias o de navegación Puede contener algunos elementos de encabezado, pero también un logotipo, un formulario de búsqueda, el nombre del autor y otros elementos
<body> representa el contenido de un documento HTML Solo puede haber un elemento <body> en un documento
<html> Representa la raíz (elemento de nivel superior) de un documento HTML, por lo que también se denomina elemento raíz Todos los demás elementos deben ser descendientes de este elemento
